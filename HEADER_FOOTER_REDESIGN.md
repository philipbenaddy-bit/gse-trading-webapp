## 🎨 Glassmorphism Details

### Background
```css
background: rgba(255, 255, 255, 0.7)
backdrop-filter: blur(16px) saturate(180%)
border: 1px solid rgba(255, 255, 255, 0.18)
```

### Dark Mode
```css
background: rgba(26, 26, 26, 0.7)
border: 1px solid rgba(255, 255, 255, 0.1)
```
